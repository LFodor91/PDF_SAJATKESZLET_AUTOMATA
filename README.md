# PDF Automata (EK / Sajátkészlet)

## Használat
1. Töltsd le ZIP-ben
2. Csomagold ki
3. Dupla katt: indit.bat
4. PDF-eket másold az Input mappába
5. Excel az Output mappában kész

## Követelmények
- Windows
- Nincs szükség Python telepítésre
