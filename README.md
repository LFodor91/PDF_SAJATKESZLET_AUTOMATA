# PDF_SAJATKESZLET_AUTOMATA

PDF alapú tételek automatikus feldolgozása Excelbe  
(Sajátkészlet verzió)

---

## Használat

1. Töltsd le a repository-t ZIP-ben (Code → Download ZIP)
2. Csomagold ki
3. Töltsd le és csomagold ki a hordozható (portable) Python környezetet
4. Dupla katt: `indit.bat`
5. A feldolgozandó PDF-eket másold az `Input` mappába
6. Az elkészült Excel az `Output` mappában jelenik meg  
   **Sajatkeszlet.xlsx** néven

Megjegyzés: a program futása közben figyeli az `Input` mappát,  
és az újonnan bemásolt PDF-eket automatikusan feldolgozza.

---

## Python környezet

A program futtatásához hordozható (portable) Python szükséges.

Hivatalos letöltési oldal:  
https://www.python.org/downloads/windows/

Itt a **Python Embedded Distribution (Windows embeddable package 64-bit)** verziót kell letölteni.
